## Project Updates

### bootc / ostree

*   **bootc v1.14.1:**  Includes a hotfix that re-attaches previously missing source artifacts and incorporates a new experimental feature to export images as tar archives (`container: Add export --format=tar command`). The `export --format=tar` feature has installation story implications by supporting a more mutable by default mode.
    *   [https://github.com/bootc-dev/bootc/releases/tag/v1.14.1](https://github.com/bootc-dev/bootc/releases/tag/v1.14.1)
*   **bootc v1.14.0:** Features include:
    *   Pre-flight disk space check before downloading updates.
    *   Display information about `/usr` overlays in `bootc status`.
    *   Significant progress on composefs productization, including SELinux enforcement for sealed images and garbage collection.
    *   Bug fixes related to SELinux relabeling and handling of `/run/systemd/resolve`.
    *   [https://github.com/bootc-dev/bootc/releases/tag/v1.14.0](https://github.com/bootc-dev/bootc/releases/tag/v1.14.0)
