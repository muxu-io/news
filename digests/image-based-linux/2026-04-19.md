### Talos

*   **Talos v1.13.0-rc.0 Release:** This release includes a kernel built with Clang, Container Device Interface (CDI) support, `talosctl debug` functionality, a new Environment Configuration Document, and support for virtiofs-based external volumes. The release contains breaking changes related to the format of EtcdConfigs, KubeletConfigs, ControllerManagerConfigs, SchedulerConfigs, or APIServerConfigs. ([v1.13.0-rc.0](https://github.com/siderolabs/talos/releases/tag/v1.13.0-rc.0))
