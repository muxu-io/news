## Executive Summary

This digest highlights several critical vulnerabilities, particularly in the AVideo platform, and potential SQL injection risks in widely-used libraries. Multiple vulnerabilities in the AVideo platform expose unauthenticated remote code execution chains. Engineers should prioritize patching AVideo deployments, securing Ory infrastructure by rotating pagination secrets, and auditing MySQL configurations when using the Kysely library.

## Critical / Actively Exploited

This includes Clone Key disclosure, database dump, and command injection. **Immediate action required: Patch AVideo deployments.**
