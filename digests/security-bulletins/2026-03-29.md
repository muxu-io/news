## Executive Summary

Multiple critical vulnerabilities affecting various open-source projects require immediate attention.  Exploitable path traversal and code injection vulnerabilities exist in Incus, Handlebars.js, and other projects.  Additionally, privilege escalation issues and SSRF vulnerabilities need to be addressed promptly to prevent unauthorized access and potential data breaches.

## Critical / Actively Exploited

*   **OpenClaw: Gateway local shared-auth reconnect silently widens paired device scope from operator.read to operator.admin and reach node RCE** (GHSA-fqw4-mph7-2vr8). Gateway local shared-auth reconnect silently widens paired device scope from operator.read to operator.admin, potentially allowing full control of stolen hosts and RCE. Update to >= 2026.3.25.
*   **Zebra node crash — V5 transaction hash panic (P2P reachable)** (GHSA-qp6f-w4r3-h8wg). Remote Denial of Service vulnerability allows unauthenticated attackers to crash Zebra nodes via crafted V5 transactions. Update to Zebra 4.3.0 or later immediately.

ChatWorkflow.chat() is vulnerable to server-side template injection via a non-sandboxed Jinja2 environment, enabling remote code execution.  Avoid passing user input directly to this method or upgrade.
*   **Incus vulnerable to arbitrary file read and write through pongo templates** (GHSA-83xr-5xxr-mh92). Instance template files can be used to cause arbitrary read or writes as root on the host server. Upgrade is required.
