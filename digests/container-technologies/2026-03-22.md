This week's updates include a new beta release of containerd (v2.3.0-beta.0) which marks the first LTS release under the new cadence aligned with Kubernetes, and includes updates to plugin config migration, CRI, image distribution, and NRI. A patch release of runc (v1.3.5) fixes issues found in v1.3.4, specifically atime-related mount flags and a regression causing spurious container start errors.
