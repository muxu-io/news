## Executive Summary

Moby saw several releases this week, including v29.3.0 of Docker Engine, which features networking and API improvements, as well as various bug fixes. BuildKit updated to v0.28.0, which includes improvements such as a new provenance format and parallelized image pushing. Podman discussions centered around auto-updating OCI artifacts and dynamically selecting user IDs for rootless containers.

## Project Updates

### Docker / Moby

*   **v29.3.0 release:** Includes several improvements and bug fixes. Key changes:
    *   Adds `bind-create-src` option to `--mount` flag for bind mounts ([docker/cli#6792](https://github.com/docker/cli/pull/6792))
    *   CLI plugin hooks now fire on command failure, allowing for error-specific hints ([docker/cli#6794](https://github.com/docker/cli/pull/6794))
    *   Lowered minimum API version from v1.44 to v1.40 (Docker 19.03) ([moby/moby#52067](https://github.com/moby/moby/pull/52067))
    *   Updates BuildKit to v0.28.0 ([moby/moby#52135](https://github.com/moby/moby/pull/52135))
    *   Fixes DNS config corruption on daemon reload ([moby/moby#52060](https://github.com/moby/moby/pull/52060))
    *   `POST /networks/{id}/connect` now correctly applies the `MacAddress` field ([moby/moby#52040](https://github.com/moby/moby/pull/52040))
    *   `GET /images/json` now supports an `identity` query parameter, including manifest summaries and trusted identity information ([moby/moby#52030](https://github.com/moby/moby/pull/52030))
    *   The `--gpus` option now uses CDI-based injection for AMD GPUs ([moby/moby#52048](https://github.com/moby/moby/pull/52048))
    *   Adds sd\_notify "RELOADING" notifications when signalling the daemon to reload its configuration ([moby/moby#52041](https://github.com/moby/moby/pull/52041))
    *   Includes fixes for `docker system prune`, `docker top`, and dockerd service registration on Windows.
*   **client/v0.3.0 release:** Includes support for the `Identity` field in ImageListOptions ([moby/moby#52030](https://github.com/moby/moby/pull/52030)) and lowers the minimum API version to v1.40 ([moby/moby#52067](https://github.com/moby/moby/pull/52067)).
*   **api/v1.54.0 release:** Adds the `identity` query parameter to `GET /images/json` ([moby/moby#52030](https://github.com/moby/moby/pull/52030)). Also includes documentation improvements and fixes.

### Podman Ecosystem

*   **Discussion:** Quadlet .artifact: Add AutoUpdate key + container/pod restart mechanism ([https://github.com/containers/podman/discussions/28217](https://github.com/containers/podman/discussions/28217)).  This proposes a feature for auto-updating containers based on OCI artifacts mounted via Quadlet. This aligns with immutable infrastructure patterns.
*   **Discussion:** Dynamic userns=keep-id:uid for container USER ([https://github.com/containers/podman/discussions/28195](https://github.com/containers/podman/discussions/28195)). This suggests a way to dynamically map the container user ID to the host user ID when using `userns=keep-id`, especially when the `USER` instruction in the Dockerfile is not explicitly set or overridden. This feature would improve the usability of rootless containers.

### BuildKit

*   **v0.28.0 release:**
    *   Updates the built-in Dockerfile frontend to v1.22.0.
    *   Switches the default provenance format to SLSA v1.0 ([#6526](https://github.com/moby/buildkit/pull/6526)). This is a significant change and may impact tools consuming BuildKit's provenance attestations.
    *   Provenance attestation can be pulled directly via Source metadata request ([#6516](https://github.com/moby/buildkit/pull/6516), [#6514](https://github.com/moby/buildkit/pull/6514), [#6537](https://github.com/moby/buildkit/pull/6537)).
    *   Parallelizes pushing result images and exporting build cache for better performance ([#6451](https://github.com/moby/buildkit/pull/6451)).
    *   Adds new Source types for accessing raw blobs from image registries and OCI layouts using identifier protocols `docker-image+blob://` and `oci-layout+blob://` ([#4286](https://github.com/moby/buildkit/pull/4286)).
    *   Supports custom checksum requests for HTTP sources ([#6527](https://github.com/moby/buildkit/pull/6527), [#6537](https://github.com/moby/buildkit/pull/6537)) and PGP signature verification for HTTP sources ([#6527](https://github.com/moby/buildkit/pull/6527)).
    *   Updates embedded Qemu emulator to v10.1.3 ([#6524](https://github.com/moby/buildkit/pull/6524)).
    *   Improves BuildKit Cgroups implementation for Kubernetes environments ([#6368](https://github.com/moby/buildkit/pull/6368)).
*   **dockerfile/1.22.0 release:** Includes fixes for linter warnings and improvements to named source sharing ([#6478](https://github.com/moby/buildkit/pull/6478)).
