## Executive Summary

The Yocto Project 6.0 Milestone 2 release is now available. Development continues towards Yocto 6.0 M3, which will be the feature freeze for the next LTS release. The meta-arm, meta-freescale-3rdparty, meta-freescale, and meta-tegra layers have been updated to be compatible with the wrynose release series. The end-of-life for the kirkstone release is approaching in April 2026, so users are encouraged to submit patches for the final release.

## Critical/Urgent

- **End of Kirkstone LTS Support:** The last release of kirkstone (2.0 forbitbake) will be 4.0.35, built in the week of April 6, 2026, for release on April 22, 2026. Submit patches by the end of March.

## Yocto Core

- **Yocto Project 6.0 Milestone 2 (M2) Released:** The second milestone release for Yocto Project 6.0 (yocto-6.0_M2) is now available for download. Includes updates to BitBake and several meta layers. Full Test Report is available.
    - Download: https://downloads.yoctoproject.org/releases/yocto/milestones/yocto-6.0_M2
- The NPM fetcher is disabled because of security concerns. See BitBake commit 355cd226e072 for more information.

## Platform Updates

### NXP i.MX

## Technical Discussions
