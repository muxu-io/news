## Executive Summary

This week's updates include preparations for the end-of-life of the Yocto Project 5.3 (Whinlatter) release, several security and feature updates to packages in meta-openembedded, and continued work on Yocto documentation. NVIDIA has released updates to its AI models and tools for the Jetson platform, and the meta-freescale and meta-arm layers have seen maintenance updates. Notably, there is a discussion around Secure Boot on Jetson AGX Orin and some compatibility issues with newer versions of glibc.

## Critical/Urgent

-   **Whinlatter End-of-Life:** The Yocto Project 5.3 (Whinlatter) release is approaching its end-of-life, with the final release (5.3.4) scheduled for the week of May 6, 2026. Patches for inclusion need to be submitted before April 10, 2026.

## Yocto Core

-   **Yocto Project 5.3.3 Release:** Yocto Project 5.3.3 (Whinlatter) is now available for download.
-   **Renaming "abort" in bitbake-setup:** Discussion continues on renaming the "abort" rebase conflict strategy in `bitbake-setup` due to potentially negative connotations. Options like "halt," "reset," "abandon," "cancel," and "attempt" are being considered.
-   **Gentoo Package List:**  A patch adding a Gentoo package list to the Yocto Project documentation was suggested to be moved to the Wiki instead due to a lack of testing infrastructure.
-   **Tagging Scheme Changes:** Clarification has been provided regarding changes to the Yocto Project's tagging scheme within Git repositories, which may impact existing CI setups.

## Platform Updates

### NVIDIA Jetson

-   **AI Model Availability:** NVIDIA is highlighting the availability of Gemma 4 for local agentic AI.
-   **Nv-monitor Tool:**  The `nv-monitor` tool now supports Jetson Orin Nano, NX, and AGX for GPU monitoring and Prometheus metric export.
-   **JetPack 6.2.2 Issues:** Multiple users report issues with JetPack 6.2.2, including GPIO control failures, Ethernet link drops, and incorrect TensorRT outputs with the VGGT model.
-   **Jetson AGX Orin Secure Boot Discussion:** A user is seeking information on enabling BootROM Secure Boot in the field on Jetson AGX Orin without requiring recovery flashing. The query concerns the possibility of triggering fuse programming steps through OTA or from within the running system.
-    **Missing r36.5 RT Kernel Packages and Docker Images:**  A user reports that the RT kernel packages and Docker images for r36.5 (JetPack 6.2.2) are unavailable, and the OTA upgrade replaces the RT kernel with the standard kernel.

### NXP i.MX

-   **Toradex HTTPS protocol change:** meta-freescale-3rdparty now uses HTTPS protocol in SRC_URIs for toradex recipes.
-   **Updated udev-rules-qoriq**: Fix for S setting has been applied.

### Raspberry Pi

-   **Raspberry Pi 4 3GB:** A new 3GB Raspberry Pi 4 has been announced, priced at $83.75.

## Layer Compatibility

-   **meta-arm and genericarm64:** A patch in meta-arm enables Trusted Firmware-A (TF-A) support for `genericarm64` for QEMU testing, but this may inadvertently introduce TF-A dependencies to genericarm64 builds.
-   **OE-Core Feature Defaults**: Patches in OE-Core now use variables for default feature filtering and machine settings. Review and add any features you do not want to use to `DISTRO_FEATURES_OPTED_OUT`.

## Technical Discussions

-   **glibc 2.43 Compatibility:** Issues with `glibc-2.43` compatibility are being addressed, and related changes are being backported to the Whinlatter and Scarthgap releases.
-   **SDMA Performance with UART**: User looking for technical guidance on configuring DMA with uarte(HSUART) on Jetson Orin with JetPack 6.2
